===============
Getting Started
===============

Oras Python is a Python client and API for OCI registry as Storage.
You should be able to easily pull and push artifacts from Python and build
your own custom interactions.
If you have any questions or issues, please `let us know <https://github.com/oras-project/oras-py/issues>`_
